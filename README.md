## 🚀 功能详细说明

### 🤖 AI 自动客服 (付费)

- **接 DeepSeek / 智谱 GLM / OpenAI / 任意 OpenAI 兼容接口** — 你自己选模型, 你自己充值, 我们不赚 token 钱
- **风格库**: 你过往的真实回复会被自动收集, AI 模仿你的口吻和用词, 不像机器人
- **多商品独立提示词**: 不同商品配不同的卖点 / 红线 / FAQ, 套餐报价表精确到 SKU
- **价格白名单硬约束**: AI 只能报你配的价格, 想自由发挥砍价 → 自动转人工（飞书通知）
- **风控关键词**: 退款 / 投诉 / 12315 / 差评 → 立即转人工, 不让 AI 误事
- **调用日志**: 每条 AI 回复都能复盘, 不满意一键加入风格库纠正
- **运行时间窗口**: 工作日/周末/夜间分别配置启用时段

### 📦 卡券自动发货 (付费)

- **批量导入卡密** (TXT / CSV / 一行一条)
- **绑定指定商品 / 规格** — 多 SKU 商品也能精准匹配发卡
- **付款瞬间触发**: 闲鱼买家付款 → 自动私信发卡密 → 标记"已发货"
- **库存预警**: 卡密剩余不足自动飞书通知补货
- **发货流水**: 每张卡发给谁、什么时候发的, 全程可查

### 🪟 多账号独立窗口

- 每个闲鱼号一个独立 窗口, 完全隔离 (Cookie / Session / 配置)
- 不限账号数量
- 任务栏按昵称区分, 一眼找到要操作的号

### 📋 订单侧边栏 + 鱼小铺工作台

- 聊到买家时右侧自动展示该买家的订单详情、物流号、退款进度
- 内嵌闲鱼官方鱼小铺, 商品发布 / 销量数据 不用打开浏览器

## ❓ 常见问题

**Q: 会被闲鱼封号吗?**
A: 软件 100% 走闲鱼官方协议, 不爬不刷, 与官方 App 行为一致. 但 AI 自动回复请合理使用, 频繁 / 异常话术仍可能触发风控, 建议配合"工作时间窗口"限制启用时段.

**Q: 支持 Mac / Linux 吗?**
A: 当前版本仅 Windows 10/11 64 位. Mac 版评估中.

**Q: 一台电脑能挂几个号?**
A: 没有数量限制, 但 5+ 个号同时挂建议 8GB 内存以上.

**Q: AI 用的是哪家?**
A: 兼容 OpenAI 接口的所有服务商: DeepSeek (推荐, 极便宜) / 智谱 GLM / OpenAI 官方 / 通义千问 / 文心一言 / Moonshot 等. 你自己注册账号自己充值, token 费用与本软件无关.

**Q: 卡密会不会被偷?**
A: 卡密本地 AES 加密存储, 仅在买家付款瞬间解密发出去一条. 软件代码已签名, 无篡改风险.

**Q: 升级新版需要重新激活吗?**
A: 不需要. 激活码绑定闲鱼账号, 软件升级不影响.
