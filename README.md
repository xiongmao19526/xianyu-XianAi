## 💎 为什么选闲AI

### 🛡️ 原生网页内核, 闲鱼"看"不出来 — 风控天花板

- **不是协议机器人, 不是 API 模拟**. 软件直接跑闲鱼官方网页 IM, 你点鼠标的所有动作软件都用同一套点法, 闲鱼后端**无法区分手动操作和软件操作**.
- 消息收发走**真实 WebSocket** (用 Chromium CDP 拦截原生帧, 不重写不伪造), 接口签名走**官方 mtop**, cookie 用**正版浏览器 session** 自动续期 — 和你拿 Chrome 浏览器开闲鱼一模一样.
- 内置**4 层防风控护甲**: ① 随机延时回复 (默认 3-15s, 可调) ② cookie 多副本自动取最新 (杜绝因 stale token 触发风控) ③ 擦亮商品/批量评价**随机间隔**, 触发限流**自动停止保护** ④ 系统休眠/网络中断自动检测与恢复, 不会"白闪"

### 🚀 多账号挂机, 一开就是全家桶

- 一台电脑挂 N 个号 (实测 5-10 个稳定, 上限取决于你的内存), **互相 100% 隔离** — 独立 cookie / 独立窗口 / 独立托盘 / 独立飞书通知, 一个号被风控不影响其它
- 每个账号一个独立的 Chromium session, **闲鱼后端看到的就是 N 个不同浏览器**, 不存在"指纹同源"问题
- 配置文件**一键导出导入**, 换电脑 30 秒搬家, 不用从头配

### 🧠 AI 不胡说 — 行业最强反幻觉方案

- 主流大模型客服都有"硬编"通病 (尺寸瞎报、价格瞎答), 闲AI 用 **JSON 结构化输出 + citation gate** 双闸门: AI 回复必须引用商品已知事实, **编造内容自动拦截转人工**
- 自带**置信度过滤**: 低置信度回复直接走人工, 不再硬猜出错单
- **凌晨自动蒸馏**: 把每个商品的历史对话提炼成卖点 / 常见问题 / 必问清单 / 适配机型, 越用越准, 不用你手写 prompt
- **风格学习**: 你手动回一次, AI 下次自动模仿你的语气, 让买家感觉一直是真人在聊

### 🔒 数据全本地, 不进作者服务器

- 所有聊天记录 / 卡密库 / API Key / 商品提示词全部存在 `%APPDATA%/xianyu-seller-im/`, **不上传任何服务器**
- Cookie 用 **AES 加密**写盘, 即使别人拿到电脑也偷不走登录态
- 唯一联网的就是软件**激活校验** (只传卡密) 和**云端更新检测** — 业务数据零外泄

### ⚡ 秒级响应 — 比同行快 N 倍

- 订单付款 → **秒级触发自动发货**, 比"几分钟刷新一次"的同类工具快几个数量级, 抢占买家好评窗口
- 自动好评在闲鱼"待评价"出现的当晚自动完成, **不会错过黄金 24h 评价时段**
- 自动擦亮主接口 + 备接口双链路, 主接口失败自动切换, **不会因为闲鱼临时改接口就停摆**

### 🛠️ 长期维护, 不是一次性卖软件

- 持续跟进闲鱼接口/风控变化, **平均每 1-2 周一次更新** (内置自动更新检测 + 红点提醒)
- 已经历 11 天 4 账号实测的稳定性打磨 (cookie 提前刷窗口、WS 健康监测、powerMonitor 唤醒静默期等), 每月 1800+ 条 warn 已优化掉 ~83%
- 客户端开源透明, 服务端独立可自托管, 不卡你脖子
